TlRadio - Telegram Radio

Setup

Requirements

  • Python3.7 or higher.
  • Postgresql.

Production setup

  1. Create a directory (mkdir tl_radio) and enter it (cd tl_radio).
  2. Install TL-Radio using pip (pip3 install -U tl_radio).
  3. Generate a configuration file using python3 -m tl_radio --generate-config.
  4. Edit the newly created config.yaml file.
  5. Run TlRadio (python3 -m tl_radio).

Upgrading

  1. Run the step 2 again.
  2. Restart TL-Radio.

Development setup

  1. Clone the repository (git clone https://nitanmarcel.com/tl_radio).
  2. Enter the cloned repository (cd tl_radio).
  3. Checkout the development branch (git checkout dev).
  4. Install dependencies (pip3 install -r requirements.txt).
  5. Make the desired changes.

Upgrading

  1. Run the step 2 again.
  2. Restart TL-Radio.
