TlRadio - Telegram Radio

Introduction

Welcome to TL-Radio docs!

This book contains usage for TL-Radio. This book has been built using mdBook.

Setup

Requirements

  • Python3.7 or higher.
  • Postgresql.

Production setup

  1. Create a directory (mkdir tl_radio) and enter it (cd tl_radio).
  2. Install TL-Radio using pip (pip3 install -U tl_radio).
  3. Generate a configuration file using python3 -m tl_radio --generate-config.
  4. Edit the newly created config.yaml file.
  5. Run TlRadio (python3 -m tl_radio).

Upgrading

  1. Run the step 2 again.
  2. Restart TL-Radio.

Development setup

  1. Clone the repository (git clone https://nitanmarcel.com/tl_radio).
  2. Enter the cloned repository (cd tl_radio).
  3. Checkout the development branch (git checkout dev).
  4. Install dependencies (pip3 install -r requirements.txt).
  5. Make the desired changes.

Upgrading

  1. Run the step 2 again.
  2. Restart TL-Radio.

Configuration

Requirements

  • Telegram App Id from my.telegram.org
  • Telegram Api Hash from my.telegram.org
  • Postgresql (sudo apt install postgresql)

1: Open config.yaml with your favorite editor. 2: Replace all the lines noted with ??? with the required values:

Telegram

  • api_id - Your App Id from my.telegram.org.
  • api_hash - Your App Hash from my.telegram.org.

General

  • sqlalchemy_db_uri - Your database url. eg: postgresql://username:password@localhost:5432/database. See section bellow on how to create a database.

Database

  1. install postgresql: sudo apt-get update && sudo apt-get install postgresql

  2. change to the postgres user: sudo su - postgres

  3. create a new database user (change YOUR_USER appropriately): createuser -P -s -e YOUR_USER

  4. create a new database table:

  5. createdb -O YOUR_USER YOUR_DB_NAME

You should now be able to build your database URI. This will be:

sqldbtype://username:pw@hostname:port/db_name

Replace sqldbtype with whichever db youre using (eg postgres, mysql, sqllite, etc) repeat for your username, password, hostname (localhost?), port (5432?), and db name.

Config Updates

The config.yaml file gets updated each time the bot has started. To view if there are any new configuration options check the changelog on github or find them in the config.yaml file.

Usage

After setting up tl_radio and running it you get a set of commands you can use in your chat. Use the help command (defaults to /help) to get a list of all the available commands.

Commands noted with *args also accepts a set of parameters. Use /{your_command} --help to get more informations about a specific command and their arguments.

Commands

  • /help - Displays a list of all commands and their short description.
  • /play *args - Downloads audio from source and adds it to the queue. If nothing is playing, play it.
  • /stop - Stops music/radio.
  • /skip - Skips to the next item in queue
  • /shuffle - Shuffles the current queue.
  • /pause - Pause the current playout.
  • /resume - Resumes the current playout.
  • /radio - *args Turns on radio and plays audio from a given stream url.
